# Yatzy

Sovelluksen avulla voidaan pelata yatzy-peliä graafisessa käyttöliittymässä. 

# Komentorivitoiminnot

## Testaus

### Ohjelman testaaminen

```
mvn test
```

### Testikattavuusraportin luominen

```
mvn jacoco:report
```
